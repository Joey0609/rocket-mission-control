# 项目指南

## 当前状态

- 已移除 Python、bat、vbs 启动链。
- 项目仅通过 npm 启动。
- 管理端为浏览器页面（需登录）。
- 游客端为浏览器页面（无需登录）。
- 项目为纯 Node.js Web 架构，不再依赖 Electron。

## 关键要求

- 倒计时统一为单一任务计时。
- 倒计时到 0 自动转为正计时。
- 禁止出现 T-00:00 与 T+00:00 双重显示。

## 入口

- 管理端: [templates/admin.html](templates/admin.html)
- 管理登录页: [templates/admin-login.html](templates/admin-login.html)
- 游客端: [templates/visitor.html](templates/visitor.html)
- 服务端: [server/httpServer.js](server/httpServer.js)
